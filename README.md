## Sobre

A Vigilância Socioassistencial é uma das funções da Política de Assistência Social, ao lado da Proteção Social e da Defesa de Direitos (LOAS, PNAS, NOB/SUAS 2012). Sua finalidade é **produzir, sistematizar, analisar e disseminar informações territorializadas** sobre situações de vulnerabilidade e risco social, subsidiando o planejamento e a gestão do Sistema Único de Assistência Social (SUAS).

Este Guia, produzido pela **Coordenação Estadual de Vigilância Socioassistencial (COOVIG)** da Secretaria de Estado de Desenvolvimento Social e Direitos Humanos (SEDSODH), apresenta um roteiro organizado em **4 blocos temáticos**:

| Bloco | Conteúdo |
|---|---|
| **1. Fundamentação** | Base normativa, conceitos e a Vigilância no SUAS |
| **2. Institucionalização** | Formalização, Portaria, Resolução CMAS e estruturação da equipe |
| **3. Disseminação e Planejamento** | Produção e disseminação de informações, apoio ao planejamento municipal |
| **4. Consolidação** | Consolidação da Vigilância como função permanente da gestão |

Inclui ainda **modelos prontos de Portaria e Resolução do CMAS**, checklists, perguntas frequentes e as **10 Regras de Ouro** para implantação.

## Executar localmente

```bash
# Clonar o repositório
git clone https://github.com/SEDSODH/guia-vigilancia-socioassistencial.git
cd guia-vigilancia-socioassistencial

# Instalar dependências
npm install

# Modo desenvolvimento
npm run dev

# Build de produção
npm run build

# Preview do build
npm run preview
```
